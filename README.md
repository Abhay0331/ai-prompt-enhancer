# ✨ AI Prompt Enhancer

A simple and powerful web tool that turns your basic ideas into highly detailed, professional prompts for AI image generators like Midjourney, Flux, Stable Diffusion, etc.

## Features

- Modern and clean dark UI
- 9 different art styles to choose from
- Powered by Llama 3.1 (via Ollama)
- One-click copy button
- Fast and easy to use

## Tech Stack

- Backend: Flask (Python)
- LLM: Llama 3.1 + LangChain
- Frontend: HTML, CSS, JavaScript

1. Install Ollama and run:
   ```bash
